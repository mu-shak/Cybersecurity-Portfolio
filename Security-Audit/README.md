# Security Audit

This folder contains security audit projects completed as part of the Google Cybersecurity Professional Certificate.

## Projects

### Botium Toys Internal Security Audit
Conducted an internal security audit for Botium Toys using the NIST Cybersecurity Framework (CSF). The audit included reviewing security controls, identifying risks and vulnerabilities, assessing compliance requirements, and providing recommendations to improve the organization's security posture.

## Skills Demonstrated

- Security Auditing
- Risk Assessment
- Compliance Review
- NIST Cybersecurity Framework (CSF)
- Security Controls Evaluation
- Security Recommendations
