# Incident Reports

This folder contains practical cybersecurity incident investigations completed as part of the Google Cybersecurity Professional Certificate.

Each report follows a structured incident response process, including identifying attacks, analyzing evidence, documenting findings, recommending security improvements, and applying industry best practices such as the NIST Cybersecurity Framework.

---

## Projects

### Network Traffic Analysis

Analyzed DNS and ICMP traffic using tcpdump to identify connectivity issues affecting a website.

### Analyze Network Attacks

Investigated a SYN Flood (DoS) attack using packet analysis and documented recommendations to improve network security.

### Apply OS Hardening Techniques

Investigated a brute-force attack that resulted in malware delivery and browser redirection. Documented the incident and recommended operating system hardening measures.

### NIST CSF Incident Report Analysis

Applied the five core functions of the NIST Cybersecurity Framework (Identify, Protect, Detect, Respond, and Recover) to analyze a Distributed Denial of Service (DDoS) attack and develop a structured incident response strategy.
