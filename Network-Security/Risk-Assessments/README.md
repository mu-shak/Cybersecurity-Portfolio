# Risk Assessments

This folder contains security risk assessments completed as part of the Google Cybersecurity Professional Certificate.

These projects focus on identifying vulnerabilities, evaluating organizational risk, and recommending practical security controls to strengthen an organization's security posture.

---

## Projects

### Network Hardening Security Risk Assessment

Evaluated network vulnerabilities and recommended security hardening practices including:

- Firewall Configuration
- Strong Password Policies
- Multi-Factor Authentication (MFA)
- Port Filtering
